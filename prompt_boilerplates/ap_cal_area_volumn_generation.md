### 数学公式规范

| 类型 | 正确写法 |
|------|----------|
| 定积分 | `$\displaystyle\int_a^b f(x)\, dx$` |
| 面积 | `$\displaystyle\int_a^b [f(x) - g(x)]\, dx$` |
| 圆盘法体积 | `$\pi\displaystyle\int_a^b [f(x)]^2\, dx$` |
| 垫圈法体积 | `$\pi\displaystyle\int_a^b \left([R(x)]^2 - [r(x)]^2\right)\, dx$` |
| 柱壳法体积 | `$2\pi\displaystyle\int_a^b x\cdot f(x)\, dx$` |
| 极坐标面积 | `$\dfrac{1}{2}\displaystyle\int_\alpha^\beta r^2\, d\theta$` |
| 分数 | `$\dfrac{a}{b}$` |

## 题目设计要求

### 通用原则

1. **难度分布**：简单 (30%) / 中等 (50%) / 困难 (20%)
2. **答案设计**：
   - 正确答案随机分布 (A-E 均衡)
   - 干扰项应基于常见错误设计
   - 避免"以上皆非"类选项
3. **数值题**：
   - 答案应为简洁的数值或表达式
   - 避免过度复杂的计算
4. **概念题**：
   - 题目表述清晰无歧义
   - 测试核心概念而非记忆

### 面积计算主题

| 主题 | 要求 |
|------|------|
| 曲线下面积 | 基本积分应用 |
| 两曲线间面积 | 水平/垂直切片选择 |
| 绝对值函数面积 | 分段处理 |
| 三角函数面积 | 周期性考虑 |
| 指数/对数函数面积 | 积分技巧 |
| 极坐标面积 | BC 专属，含玫瑰曲线、心形线等 |

### 体积计算主题

| 主题 | 要求 |
|------|------|
| 圆盘法 | 绕 x 轴或 y 轴旋转 |

| 绕非坐标轴旋转 | 如 y = k, x = h |
| 正方形截面 | 边长由区域宽度决定 |
| 半圆/等边三角形截面 | 面积公式需正确应用 |
| 等腰直角三角形截面 | BC 专属 |

### 简答题示例

```latex
\noindent\textbf{Question 1}\\[0.5em]

Let $R$ be the region bounded by the graphs of $y = \ln x$, $y = 2$, and $y = 0$.

\begin{enumerate}[label=\alph*.]
    \item Find the area of region $R$.
    \item Find the volume of the solid generated when $R$ is revolved about the $x$-axis.
    \item Write, but do not evaluate, an integral expression for the volume of the solid generated when $R$ is revolved about the line $x = -1$.
    \item Write, but do not evaluate, an integral expression for the volume of the solid whose base is $R$ and whose cross-sections perpendicular to the $x$-axis are squares.
\end{enumerate}
```

```latex
\noindent\textbf{Question 2}\\[0.5em]

\textbf{(BC Only)} Consider the region $R$ bounded by the polar curves $r = 2$ and $r = 4\cos\theta$.

\begin{enumerate}[label=\alph*.]
    \item Find the points of intersection of the two curves.
    \item Find the area of the region that lies inside both curves.
    \item Set up, but do not evaluate, an integral expression for the area of the region that lies inside $r = 4\cos\theta$ but outside $r = 2$.
    \item For the curve $r = 4\cos\theta$, find $\dfrac{dy}{dx}$ at the point where $\theta = \dfrac{\pi}{3}$.
\end{enumerate}
```
