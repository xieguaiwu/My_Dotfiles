# 文献阅读辅助

## 任务目标
阅读指定文献，查找特定内容的原始出处，并提供详细的引用信息以便验证和引用。

## 执行流程

### 1. 读取文献
读取指定的文献文件（支持 PDF、Markdown 等格式）。

### 2. 分析目标内容
根据 `target_content` 描述，识别需要查找的关键信息：
- 关键概念或术语
- 具体结论或定理
- 实验方法或数据
- 历史事件或人物

### 3. 在文献中定位相关内容

**搜索策略**：
- 使用关键词搜索
- 查找参考文献列表
- 跟踪引用链

**定位信息**：
- 所在章节
- 页码
- 段落位置

### 4. 查找原始出处

#### 内部引用查找
- 提取文献中的引用标记（如 `[1]`, `(Smith, 2020)`）
- 匹配参考文献列表

#### 外部搜索
使用 `websearch_web_search_exa` 搜索：
```
"{关键内容}" original source
"{作者}" "{年份}" {主题}
site:scholar.google.com {关键词}
```

#### 获取原文
使用 `webfetch` 获取原文内容：
- DOI 链接
- 学术数据库
- 作者主页

### 5. 验证引用信息

确认以下信息：
- 作者姓名拼写
- 发表年份
- 期刊/会议名称
- 卷号、期号、页码
- DOI 或 URL

### 6. 生成报告

## 输出格式

```markdown
## 文献引用查证报告

### 查询内容
{target_content}

### 找到的出处

#### 出处 1
- **原文**: "{原文引用内容}"
- **来源**: {作者}. {标题}. {期刊/会议}, {年份}, {卷}({期}): {页码}.
- **DOI**: {DOI链接}
- **页码**: 第 {X} 页
- **原文意义**: {中文解释}

#### 出处 2
...

### 引用建议

**GB/T 7714 格式**:
```
{作者}. {标题}[J]. {期刊}, {年份}, {卷}({期}): {页码}.
```

**APA 格式**:
```
{Author} ({Year}). {Title}. {Journal}, {Volume}({Issue}), {Pages}.
```

### 验证状态
- [x] 原文已找到
- [x] 页码已确认
- [x] 内容含义已解释
```

## 注意事项

1. **准确性优先**: 确保引用信息准确无误
2. **页码精确**: 提供具体页码便于验证
3. **原文语言**: 如原文非英文，说明原文语言
4. **访问限制**: 如原文需要付费访问，注明获取方式
5. **多重引用**: 如内容有多个出处，逐一列出
6. **未找到情况**: 如无法找到原文，说明可能原因并建议替代方案

## 常见问题

### Q: 文献中的引用格式不标准怎么办？
A: 尝试多种搜索方式：
- 作者 + 年份 + 关键词
- 标题关键词搜索
- DOI 查询

### Q: 原文是付费论文怎么办？
A: 可以：
- 通过学校图书馆访问
- 查找作者主页预印本
- 使用 arXiv 等开放平台
- 使用 Scholar.google.com 查找引用该论文的后续研究
