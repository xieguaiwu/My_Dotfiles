# 翻译辅助

## 任务目标
将外文论文翻译为中文 Markdown 文档，保持数学公式的正确性和文档结构的一致性。

## 执行流程

### 1. 读取源文件
使用 `read_file` `look_at` 或者`pdftotext`等方式 读取源文件。

### 2. 分析文档结构
- 识别标题层级（Abstract, Introduction, Methods 等）
- 提取章节结构
- 标记数学公式、图表、代码块等特殊内容

### 3. 翻译处理

#### 标题翻译 （英文）
| 英文 | 中文 |
|------|------|
| Abstract | 摘要 |
| Introduction | 引言 |
| Methods / Methodology | 方法 |
| Results | 结果 |
| Discussion | 讨论 |
| Conclusion | 结论 |
| References | 参考文献 |

#### 正文翻译原则
- **准确性**: 保持原文含义，不随意增删
- **流畅性**: 符合中文表达习惯
- **专业性**: 使用标准学术术语
- **一致性**: 同一术语全文统一翻译

#### 特殊内容处理
- **数学公式**: 保持 LaTeX 格式，不翻译变量名
- **专有名词**: 首次出现时可标注英文原文
- **人名/地名**: 保留原文或使用通用译名
- **代码**: 保持原样不翻译

### 4. 数学公式转换

确保所有数学表达式使用正确的 LaTeX 格式：

| 类型 | 格式 |
|------|------|
| 行内公式 | `$E = mc^2$` |
| 块级公式 | `$$\int_a^b f(x) \, dx$$` |
| 多行公式 | 使用 `align` 环境 |

**注意事项**：
- 保留原始公式的所有符号
- 不翻译公式中的外文单词（如 `\sin`, `\log`）
- 检查括号配对

### 5. 输出文件

**文件命名**: `AI中译{论文的外文名}.md`

**输出格式**:
```markdown
# {中文标题}

> 原文：{外文标题}
> 翻译日期：{日期}

## 摘要
{翻译内容}

## 1. 引言
{翻译内容}

...
```

### 6. 质量检查

- [ ] 所有章节已翻译
- [ ] 数学公式格式正确
- [ ] 专业术语翻译一致
- [ ] 无遗漏段落
- [ ] 参考文献已省略（按用户要求）

## 输出格式

```markdown
# {论文中文标题}

> 原文标题：{Original Title}
> 原文作者：{Authors}

## 摘要

{摘要翻译内容}

**关键词**：{关键词翻译}

## 1. 引言

{引言翻译内容}

## 2. 方法

{方法翻译内容}

...

---

*翻译说明：本文为AI辅助翻译，仅供参考。*
```

## 注意事项

1. **省略参考文献**: 按要求不翻译参考文献列表
2. **公式完整性**: 确保所有公式符号完整、正确
3. **术语一致性**: 建议维护术语对照表
4. **长句处理**: 适当拆分长句，保持中文流畅
5. **被动语态**: 英文被动语态可转为中文主动语态
6. **时态转换**: 学术论文常用现在时，翻译时注意语境
